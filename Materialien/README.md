# Materialien
Dieser Ordner enthält diverse **Materialen zur Nutzung im Unterricht**, wie etwa Grafiken, Dokumente, usw.  

Die Materialien sind nicht sortiert, sondern werden hier nur vorgehalten, damit sie von den anderen Dokumenten in diesem Repository referenziert werden können.
